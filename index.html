Heny's log

안녕하세요! 장혜은입니다.

새로운 것에 대한 흡수가 빠르고 디테일을 놓치지 않는 꼼꼼함을 겸비한
웹 퍼블리셔입니다.

자신의 일을 사랑하고 즐기는 자에게 성공과 발전이 따라온다고 생각합니다. 어떤 상황에서든 즐거움을 바탕으로 앞으로 나아가기 위해 노력해요.

저의 가치를 녹여내 더욱 가치있는 화면을 구현하고자 합니다.

Education

  • 하이미디어 과정평가형 UX/UI 디자이너 웹 퍼블리셔 훈련과정 수료

  • 추계예술대학교 실용음악과 졸업

  • 청주여자고등학교 졸업

Certification

  • 웹디자인기능사 한국산업인력공단

Mark up

콘텐츠와 컨셉에 맞는 태그를 사용하여 구현합니다. 또한 상황에 따라 bootstrap, aos와 같은 라이브러리를 적절하게 활용할 수 있어요.

  • HTML5 마크업

  • 웹 표준성 / 웹 접근성 준수

  • 크로스 브라우징

  • 상황에 맞는 시맨틱태그 사용

  • Bootstrap, aos 등 라이브러리 활용

styles

css는 물론 scss를 통해 견고한 디자인을 하고자 합니다. UI/UX 사용성을 고려한 반응형 레이아웃을 구현합니다.

  • CSS3

  • 반응형 레이아웃

  • Sass 변수 등 활용

  • BEM 방법론 적용

javascript / jQuery

Javascript 문법을 이해하고 활용하여 다양한 기능을 구현할 수 있습니다. 현재에 안주하지 않고 더욱 좋은 기능을 섬세하게 구현할 수 있도록 계속해서 탐구하고 있습니다.

  • Vanilla JavaScript

  • 라이브러리 소스 활용

Co-work

동료들과 협업해 결과물을 만들어내는 것에 보람을 느낍니다. Git 등 형상 관리 툴을 활용하여 협업할 수 있어요. 기본적인 원칙을 지키며 그 안에서 수용적인 자세로 동료들과 함께 옳다고 생각하는 것을 이루어나갑니다.

  • Git

  • Source tree

삼성반도체 (유지/보수)

html css 반응형 javascript

저의 첫 프로젝트입니다. 입사 후 처음 맡은 업무로서 무엇보다도 협업에 대해 배울 수 있었습니다. 협업 툴을 사용하여 업무내용을 공유하고 명확하고 간결하게 전달해야 하는 것, 그리고 다른 부서와의 의사소통에서 쓰이는 용어들에 대한 중요성을 느꼈습니다. 또한 동시에 진행되는 업무가 많은 경우 업무 중요도를 파악하여 순차적으로 진행하는 등 업무방식에 대해 고려해볼 수 있는 경험이었습니다.
