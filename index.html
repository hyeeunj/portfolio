about

안녕하세요! 풀스택 개발자 장혜은입니다.

약 3년간 웹 퍼블리셔로서 삼성반도체, 한화에어로스페이스 등 다양한 기업 사이트를 구축하며 쌓은 프론트엔드 역량을 기반으로, 백엔드와 AI까지 아우르는 풀스택 개발자로 성장하고 있습니다.

Next.js, React, TypeScript를 활용한 프론트엔드 개발은 물론, Spring Boot, FastAPI, Docker, AWS 등을 활용한 서버 아키텍처 설계와 AI Agent 서비스 개발까지 프로젝트의 전체 흐름을 이해하고 구현합니다.

사용자가 마주하는 화면부터 그 뒤의 데이터 흐름, 그리고 AI가 판단하는 비즈니스 로직까지 하나의 시선으로 바라볼 수 있는 것이 저의 가장 큰 강점입니다.

핵심 내용

대규모 트래픽과 다중 권한 처리를 소화하는 안정적인 서비스 구조 설계 및 유연한 권한 검증 로직 구현

구현 기능 및 개발 과정

  • 01
    프로젝트 리드 및 데이터 모델링
    • 초기 WBS(업무 분할 구조도) 정의를 통한 전체 타임라인 관리 주도
    • 대규모 트래픽 대비 확장성과 무결성을 고려한 PostgreSQL 기반 ERD 모델링 수행
  • 02
    유저 플로우 및 아키텍처 설계
    • 탐색, 티켓 예매, 알림으로 이어지는 전체 유저 저니(User Journey) 백엔드 로직 구현
    • 도메인 간 데이터 흐름의 독립성을 보장하는 유연한 아키텍처 구조 설계
  • 03
    컨테이너 기반 인프라 구축
    • 독립적인 서비스 구동을 위해 각 서비스를 Docker 컨테이너로 분리
    • Nginx 리버스 프록시 설정을 통한 외부 요청 라우팅 최적화 환경 구축

성과

  • 명확한 ERD 및 WBS 설계를 기반으로 대규모 요구사항 정의를 체계적으로 완료
  • 유연하고 확장성 있는 권한 시스템을 성공적으로 안착시켜, 향후 새로운 이벤트 형태나 권한 등급이 추가되더라도 코드 수정 없이 유연하게 대응할 수 있는 아키텍처 완성

느낀 점

대규모 개발 요구사항을 효과적으로 충족하기 위한 ERD 및 WBS 설계 역량을 기를 수 있었습니다. 아키텍처 설계 단계에서 구조를 촘촘히 다지는 것이 팀 프로젝트의 생산성을 대폭 향상시킨다는 점을 깊이 깨달았습니다.

담당 역할

인프라 구축 / AI 비즈니스 로직 설계 / 권한 인증 체계 구축 / 컨시어지 에이전트 개발

Tech Stack
#Next.js #React #TypeScript #Zustand #Spring Boot #Spring Security #JWT #FastAPI #Google Gemini API #PostgreSQL #Neo4j #Redis #AWS #Docker #GitHub Actions
링크
홈페이지 GitHub

핵심 내용

AI Agent의 정교한 비즈니스 로직 설계와 역할 기반 접근 제어(RBAC) 권한 인증을 통해, 보안성과 효율성을 갖춘 호텔 백오피스 자동화 시스템 구축

구현 기능 및 개발 과정

  • 01
    부서별 AI 에이전트 개발 및 시스템 구축
    • 고객 요청의 문맥을 분석하여 우선순위 지정 및 담당 부서 자동 라우팅 시스템 구축
    • 호텔 도메인에 부합하는 판단을 위해 시스템 프롬프트(System Prompt) 및 페르소나 가이드라인 고도화
  • 02
    역할 기반 접근 제어(RBAC) 권한 인증 설계
    • 투숙객, 프론트, 하우스키핑 등 유저 그룹별 접근 API 및 기능의 철저한 분리
    • 인증된 사용자만 백오피스 시스템에 안전하게 접근할 수 있도록 인가 플로우 구현

성과

  • 역할 기반 권한 인증 설계를 도입하여 백오피스 시스템 내 정보 유출 및 오작동 위험을 사전에 방지하는 안정성 확보
  • 비정형 고객 요청에 대해 수동 개입을 최소화하고, 권한에 맞는 담당자에게 업무가 정확히 배정되는 지능형 자동 배정 프로세스 구현
  • 프롬프트 엔지니어링을 통해 AI 에이전트의 오답률(Hallucination)을 대폭 낮추고 실무 시나리오 수준의 안정적인 응답 속도 확보

느낀 점

AI 에이전트 기술 실무 적용 시, 정교한 AI 로직만큼이나 이를 뒷받침하는 '안전한 권한 인증 설계'가 시스템 완성도를 결정한다는 것을 깊이 깨달았습니다. 유저 플로우별 권한 제어를 직접 구현하며 보안성과 효율성을 갖춘 백오피스 아키텍처 설계 안목을 길렀습니다.
